XG BOOST:¶

En este notebook vamos a tratar un modelo Boosting¶

El XGboost (Extra Gradient boosting) parte del rbol de decisiones que se implementa en minera de datos para clasificar o pronosticar sobre una variable objetivo (y), a travs, del aprendizaje automtico que se realiza sobre un set de datos, utilizando varios clasificadores dbiles que en este caso, como ya mencionamos, son nuestros rboles de decisiones, pero potenciando los resultados de estos, debido al procesamiento secuencial de la data con una funcin de prdida o coste, la cual, minimiza el error iteracin tras iteracin, hacindolo de esta manera, un pronosticador fuerte.

Sin embargo, como siempre esto va a depender del nivel de ajuste de los parmetros que utilicemos en la funcin: xgb, ya que, determinar los resultados del modelo.

As pues, indicamos las principales ventajas de utilizar este mtodo, en nuestros proyectos:

  • Trabaja sobre grandes bases de datos con mltiples variables, tanto categricas como numricas.
  • Admite missing values, o valores perdidos en medio del procesamiento del algoritmo.

No obstante presenta la siguiente desventaja principal:

  • Utiliza grandes cantidades de memoria RAM, por lo que, es recomendable procesar el algoritmo determinando previamente mediante una validacin cruzada, las variables que ms aportan informacin a la estructuracin del modelo.

Definimos y entrenamos el modelo XGBOOST.

Los modelos Gradient Boosting estn formados por un conjunto de rboles de decisin individuales, entrenados de forma secuencial, de forma que cada nuevo rbol trata de mejorar los errores de los rboles anteriores. La prediccin de una nueva observacin se obtiene agregando las predicciones de todos los rboles individuales que forman el modelo.

Matriz de Confusión:¶

La matriz de confusin de un problema de clase n es una matrz nxnen la que las filas se nombran segn las clases reales y las columnas, segn las clases previstas por el modelo. Sirve para mostrar de forma explcita cundo una clase es confundida con otra

Accuracy: Es el porcentaje total de elementos clasificados correctamente.

In [11]:
# definimos la matriz de confusin para cualquir modelo
def conf_matrix(Y_test, Y_pred): # va a comparar los resultados de las predicciones de la variable target vs la real del testset
    cnf_matrix_tra = confusion_matrix(Y_test, Y_pred)
    conf_plot = sns.heatmap(cnf_matrix_tra, annot=True, cmap='Blues', fmt='g')
    bottom, top = conf_plot.get_ylim()
    conf_plot.set_ylim(bottom + 0.5, top - 0.5)
    conf_plot.set_title("Confusion matrix, without normalization")
    conf_plot.set_ylabel('True Label')
    conf_plot.set_xlabel('Predicted Label');

def conf_matrix_norm(Y_test, Y_pred): # va a comparar los resultados de las predicciones de la variable target vs la real del testset
    cnf_matrix_tra = confusion_matrix(Y_test, Y_pred,  normalize='true')
    conf_plot = sns.heatmap(cnf_matrix_tra, annot=True, cmap='Blues', fmt='g')
    bottom, top = conf_plot.get_ylim()
    conf_plot.set_ylim(bottom + 0.5, top - 0.5)
    conf_plot.set_title("Confusion matrix, with normalization")
    conf_plot.set_ylabel('True Label')
    conf_plot.set_xlabel('Predicted Label');

Curva ROC:¶

Una curva ROC (curva de caracterstica operativa del recepto) es un grfico que muestra el rendimiento de un modelo de clasificacin en todos los umbrales de clasificacin. Esta curva representa dos parmetros:

  • Tasa de verdaderos positivos
  • Tasa de falsos positivos

El AUC proporciona una medicin agregada del rendimiento en todos los umbrales de clasificacin posibles. Una forma de interpretar el AUC es como la probabilidad de que el modelo clasifique un ejemplo positivo aleatorio ms alto que un ejemplo negativo aleatorio.
